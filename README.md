# Amazon Sales Analysis using Python and EDA

## Project Overview

This project analyzes Amazon sales data using Python and Exploratory Data Analysis (EDA). The objective is to understand sales performance, identify top-performing categories, analyze customer distribution, and generate business insights.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset Information

* Total Records: 128,975
* Total Columns: 24

---

## Data Cleaning

The following preprocessing steps were performed:

* Removed unnecessary columns
* Converted Date column to datetime format
* Checked missing values
* Checked duplicate records
* Created Month, Year, and Day features

---

## Exploratory Data Analysis

The analysis focused on:

* Revenue by Category
* Revenue by State
* Monthly Revenue Trends
* Order Status Distribution
* Quantity Sold by Category
* Fulfillment Analysis

---

## Key Findings

### Revenue Insights

* Total Revenue: ₹7.86 Crore
* Set category generated the highest revenue
* Kurta and Western Dress were the next highest revenue contributors

### Geographic Insights

* Maharashtra generated the highest revenue
* Karnataka ranked second
* Telangana ranked third

### Fulfillment Insights

* Around 70% of orders were fulfilled by Amazon
* Around 30% were fulfilled by Merchants

### Order Status Insights

* Cancellation rate was approximately 14%
* Most orders were successfully shipped and delivered

---

## Business Recommendations

* Focus marketing efforts on Set and Kurta categories.
* Increase customer acquisition in Maharashtra and Karnataka.
* Investigate causes of order cancellations.
* Improve inventory planning for top-selling products.
